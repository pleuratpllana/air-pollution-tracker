# Air Pollution Tracker – Kosovo
Live air quality dashboard for major cities in Kosovo.

Air Pollution Tracker is a real-time dashboard built to monitor air quality (AQI) across Kosovo’s main cities. It helps residents stay informed about pollution levels, visualize trends, and make daily decisions to protect health — all in a clean, minimalist interface.

**Free forever. No plans or expiring trials.**

## What Problem Does Air Pollution Tracker Solve?
Air quality in urban areas can fluctuate drastically due to traffic, industry, and weather conditions. Many residents lack a simple, reliable way to check current pollution levels and trends for their city. Air Pollution Tracker provides a single, accessible platform to monitor AQI in real time, visualize patterns over time, and make better choices for health and safety.

## Features

### Dashboard
- Real-time AQI for each of Kosovo’s 7 main cities: Prishtina, Prizren, Pejë, Gjakovë, Ferizaj, Mitrovicë, Gjilan
- Color-coded indicators for quick recognition of air quality levels
- Trend charts showing PM2.5 values over recent days
- Responsive grid layout for any device

### Insights & Trends
- Interactive charts using Chart.js
- Historical trend visualization for pollution monitoring
- City-specific snapshots to track local conditions

### Design & Usability
- Minimalist, clean interface suitable for portfolios or public dashboards
- Mobile-first design with responsive Tailwind CSS styling
- Accessible color coding for quick comprehension

## Tech Stack
- **Next.js** – Server-side rendering and routing for fast performance
- **React + TypeScript** – Component-based architecture with type safety
- **Tailwind CSS** – Utility-first responsive styling
- **Chart.js + react-chartjs-2** – Interactive trend charts
- **Fetch API** – Pulling real-time AQI data from Kosovo air quality sources
