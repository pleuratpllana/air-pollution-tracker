# Air Pollution Tracker – Kosovo

**Live air quality dashboard for major cities in Kosovo**, built with **React, TypeScript, Next.js, Tailwind CSS, and Chart.js**.  

This project displays real-time air quality (AQI) data and visual trends for the **7 main cities of Kosovo**: Prishtina, Prizren, Pejë, Gjakovë, Ferizaj, Mitrovicë, Gjilan.  

---

## Features

- Real-time **AQI data** for each city  
- **Color-coded indicators** for air quality levels  
- **Trend charts** showing PM2.5 over the last days  
- Responsive **grid layout** with Tailwind CSS  
- Clean, minimalist design suitable for portfolio  

---

## Technologies

- **Next.js** – React framework for server-side rendering and routing  
- **React + TypeScript** – component-based architecture with type safety  
- **Tailwind CSS** – utility-first styling for responsive layout  
- **Chart.js + react-chartjs-2** – for interactive trend charts  
- **Fetch API** – pulling real-time AQI data from Kosovo air quality API
